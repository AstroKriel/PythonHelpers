This project currently uses the following submodules as editable installs:
	- jormi: installed from local path: .
		Uninstall with: `uv pip uninstall jormi` # from the project root

To make this project standalone, do the following:
	1. Uninstall each local editable install using the commands mentioned above.
	2. Add the following entries to your [project.dependencies] section in pyproject.toml:
		- `jormi @ git+https://github.com/AstroKriel/PythonTools`

Note: Run the uninstall commands from the project root so that uv uses the correct environment.
